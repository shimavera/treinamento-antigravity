Uma ferramenta de IA que roda no terminal. Voce conversa com a IA e ela escreve codigo, cria arquivos, roda comandos — tudo pelo chat.

💬

Conversa Natural

⚡

Terminal Integrado

Parte 1

Antigravity = Claude Code

O Antigravity usa o Claude Code por baixo — o agente de codigo da Anthropic.

Como instalar

# 1. Instalar via npm
npm install -g @anthropic-ai/claude-code

# 2. Entrar na pasta do projeto
cd meu-projeto

# 3. Iniciar o Claude
claude

O que ele faz

  • Escreve codigo completo
  • Cria e edita arquivos
  • Roda comandos no terminal
  • Faz deploy automatico
  • Conecta com servicos externos (MCPs)
  • Usa Skills especializadas
Parte 2

O que sao Skills?

Skills sao "manuais de instrucao" que ensinam o Claude a fazer tarefas especificas com mais qualidade. E como dar um livro de receitas para um chef.

📖

Conhecimento Especializado

Cada skill contem instrucoes detalhadas sobre um assunto (n8n, Evolution API, etc)

🔌

Facil de Instalar

Um comando e a skill ja esta disponivel: /install-skill

🎯

Ativacao Automatica

O Claude detecta quando precisa de uma skill e ativa sozinho

Parte 2

O que sao MCPs?

MCP = Model Context Protocol. Sao "maos" que o Claude usa para interagir com servicos reais — banco de dados, WhatsApp, automacoes.

Voce
💬 Chat
→
Claude
🧠 IA
→
MCP
🔌 Ponte
→
Servico
🗄️ Supabase

Sem MCP

  • "Abre o painel do Supabase"
  • "Copia essa query e roda la"
  • "Me mostra o resultado"

Com MCP

  • "Claude, quantos registros tem na tabela users?"
  • → Resposta instantanea com dados reais
Parte 2

Instalando Skills e MCPs

Skills

# No Claude Code, digite:
/install-skill URL-da-skill

# Ou adicione manualmente em
# ~/.claude/skills/nome/SKILL.md

Skills sao arquivos .md com instrucoes — o Claude le e aprende

MCPs

# Configurar em:
# ~/.claude/settings.json

"mcpServers": {
  "supabase": {
    "command": "npx",
    "args": ["..."]
  }
}

MCPs rodam como processos — conectam a servicos reais

Parte 3 — Conceitos

O que e uma API?

API = "garcom do restaurante". Voce (app) faz um pedido, o garcom (API) leva ate a cozinha (servidor) e traz a resposta.

Seu App
📱 Pedido
→
API
🔗 Garcom
→
Servidor
🍳 Cozinha
→
Resposta
📦 Dados

API Key

Sua "credencial" — como um cracha que prova que voce tem permissao de usar o servico

Webhook

"Notificacao automatica" — quando algo acontece, o servico avisa voce automaticamente

Execucao

Workflow 1 — Coletor

  • Criar tabela group_messages no Supabase
  • Criar workflow no n8n com Webhook trigger
  • Configurar webhook na Evolution API
  • Filtrar: ignorar msgs proprias (fromMe === false)
  • Extrair: grupo, remetente, texto, timestamp
  • Salvar no Supabase via node Postgres/HTTP
  • Testar: mandar msg no grupo e ver no banco
Execucao

Workflow 2 — Resumo

  • Criar workflow com Schedule Trigger (cron)
  • Buscar msgs do dia no Supabase
  • Agrupar mensagens por grupo
  • Montar prompt para a Claude API
  • Chamar Claude API (node HTTP Request)
  • Enviar resumo via Evolution API
  • Marcar mensagens como processadas
  • Testar o fluxo completo
